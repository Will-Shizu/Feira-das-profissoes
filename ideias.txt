O que vai ter no site?

1- As matérias do curso do 1° ao 3° ano com um breve resumo do que se trata.
2- Fotos e texto em relação aos eventos que o curso realiza e participa.
3- Uma página para os alunos enviarem suas dúvidas.
4- Currículo dos professores da base técnica.

O que vai ter na home?

1- Uma mensagem de bem vindo.
2- Um mini-texto falando o pra que serve a página.
3- redirecionamentos pras outras páginas.

Banco de dados (Localhost):

table com nome duvidas:
	id int unsingned
	nome varchar(30)
	email varchar(100)
	pergunta varchar(200)
	respondido tinyint(1)

SQL:
	create table duvidas(id int unsigned not null primary key auto_increment, nome varchar(30) not null, email varchar(100) not null, pergunta varchar(200) not null, respondido tinyint(1) unsigned not null default 0) default character set utf8 collate utf8_general_ci;


table com nome usuarios:
	id int unsigned
	user varchar(30)
	senha varchar(200)

SQL:
	create table usuarios(id int unsigned not null primary key auto_increment, user varchar(30) not null, senha varchar(200) not null) default character set utf8 collate utf8_general_ci;
